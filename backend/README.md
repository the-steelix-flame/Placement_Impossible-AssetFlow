# AssetFlow API

Django 5 + Django Ninja backend for AssetFlow (Enterprise Asset & Resource Management).
Runs as a Hugging Face **Docker Space** on port 7860, backed by Supabase Postgres.

- Interactive API docs: `/api/docs`
- Health check: `/health`
- Deployment guide: [DEPLOY.md](DEPLOY.md)

The YAML header above is required by Hugging Face Spaces (Docker SDK, `app_port: 7860`).
